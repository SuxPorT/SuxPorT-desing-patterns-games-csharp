# Design Patterns

Project based on the book ["Design Patterns com C# - Aprenda padrões de projeto 
com os games"](https://www.casadocodigo.com.br/products/livro-design-paterns-csharp), 
by Rodrigo Gonçalves Santana, with the objective of studying the different 
Design Patterns in C# based on games.
